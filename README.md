# YT-DISTRACTION

YT-DISTRACTION is a Chrome extension designed to help you focus on educational content on YouTube by hiding distractions like recommended videos, shorts, and unrelated suggestions. Perfect for students, learners, and anyone who wants a more productive YouTube experience.

---

## Features

- Hide non-educational recommendations  
- Focus only on educational videos  
- Clean YouTube interface without distractions  
- Lightweight and easy to use  

---

## Installation

1. Clone this repository:  
   ```bash
   git clone https://github.com/GURUMITHRAN-15/YT-DISTRACTION.git
2.Open Chrome and go to chrome://extensions/

3.Enable Developer mode (top right corner)

4.Click Load unpacked and select the cloned project folder

5.The extension is now active on YouTube

## Usage

-Once installed, the extension will automatically filter YouTube recommendations.
